# 📸 ГАЙД: Як додати фото на сайт

## 🎯 5 ФОТО З ЧАТУ → САЙТ

### Крок 1: Знайдіть фото в чаті Cursor

Ви надіслали 5 фото:
1. **Фасад** - вулиця з вивіскою NEXX GSM
2. **Приймання** - менеджер у жилеті приймає телефон
3. **Робоче місце** - мікроскоп, паяльна станція
4. **Діти тренінг 1** - великий стіл з планшетами
5. **Діти тренінг 2** - дівчинка фотографує телефоном

---

### Крок 2: Збережіть кожне фото

**Для кожного фото:**

1. **Клік ПКМ** (права кнопка миші) на фото
2. **"Зберегти зображення як..."**
3. **Виберіть папку:**
   ```
   C:\NEXX GSM\images\
   ```
4. **Назвіть файл:**
   - Фото 1 → `facade.jpg`
   - Фото 2 → `reception.jpg`
   - Фото 3 → `workspace.jpg`
   - Фото 4 → `kids-training-1.jpg`
   - Фото 5 → `kids-training-2.jpg`

---

### Крок 3: Перевірте папку

```
C:\NEXX GSM\images\
├── facade.jpg              ✓
├── reception.jpg           ✓
├── workspace.jpg           ✓
├── kids-training-1.jpg     ✓
└── kids-training-2.jpg     ✓
```

Всі 5 файлів мають бути в папці!

---

### Крок 4: Оновіть сайт

1. Відкрийте:
   ```
   https://click-combine-hose-double.trycloudflare.com
   ```

2. Натисніть **Ctrl+F5** (hard refresh)

3. Фото з'являться автоматично!

## ❓ Якщо щось не так

**Фото не показується?**
- Перевірте назву файлу (має бути точно як вказано)
- Перевірте формат (має бути .jpg)
- Оновіть сторінку (Ctrl+F5)

**Фото занадто велике?**
- Оптимізуйте через https://tinypng.com
- Цільовий розмір: 300-500 KB
