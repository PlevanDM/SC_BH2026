## 🚀 Швидкий Старт Тестування

1. Відкрий `http://localhost:8080` (або публічний URL)
2. Пройдися по кожному пункту чек-листу
3. Відмічай галочки ✅ або ❌
4. Записуй знайдені баги в розділ "Нотатки"
5. Перевір на різних пристроях/браузерах

**Для публічного доступу:**
```powershell
# Встановити localtunnel (один раз)
npm install -g localtunnel

# Запустити туннель
lt --port 8080 --subdomain sc-bucharest-test

# Або через npx (без встановлення)
npx -y localtunnel --port 8080
```
